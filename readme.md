项目架构：
本项目是一套基于lora的物联网传输架构，由节点搭载ph，tds，浊度。水温等传感器和oled显示屏幕。经由lora模块的自定义数据帧发送至网关模块，网关汇总解析各节点发送过来的数据发送至onenet云平台。在与平台对接的可视化app上显示各节点在线状态，节点对应传感器数据，以及控制各节点外设工作方式。
